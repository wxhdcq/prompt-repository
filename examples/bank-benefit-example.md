# 银行卡权益检查示例

## 示例用途

演示如何用 [银行卡权益检查模板](../prompts/04-bank-card-benefit-check.md) 检查中信 GBA 卡、汇丰 Pulse、恒生香港活动。以下内容只提供查询框架，不代表真实活动。

## 调用步骤

1. 打开 [银行卡权益检查模板](../prompts/04-bank-card-benefit-check.md)。
2. 复制「完整提示词」代码块。
3. 替换目标银行或卡片名称。
4. 只填写公开活动需求，不提供卡号、验证码或身份证件。

## 可复制示例

```text
请按“银行卡权益检查模式”帮我查这张卡或这些银行活动：

目标：
中信 GBA 卡 / 汇丰 Pulse / 恒生香港

我关心：
- 新活动
- 返现
- 迎新
- 保级
- 绑定支付
- 云闪付/Apple Pay/支付宝/微信支付限制
- 活动截止日
- 是否需要报名
- 是否有地区限制
- 是否值得参加

请按以下来源优先级：
1. 银行官方网页
2. 官方活动条款 PDF
3. 银行 App 公告
4. 客服说明
5. 社区经验帖
6. 第三方整理文章

请区分：
- 官方确定信息
- 社区经验
- 推测
- 我需要自己进 App 确认的内容

输出：
1. 当前可用活动
2. 即将结束活动
3. 返现/积分规则
4. 参加门槛
5. 是否需要报名
6. 最容易踩坑的地方
7. 对我是否值得
8. 下一步操作清单

注意：
不要把社区经验当官方结论。
不要保证一定成功。
银行卡权益以官方条款和实际账户显示为准。
```

## 使用提醒

- 不要提供银行卡号、CVV、验证码、身份证件或完整账单。
- 银行 App 内个人账户显示的资格可能和公开活动不同。
- 第三方整理文章只能作为线索，不能替代官方条款。
