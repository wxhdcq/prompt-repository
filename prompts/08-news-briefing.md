# 新闻简报模板

## 用途

用于全球政治、AI、美股、德国、东京、拜仁、足球等主题新闻简报。

## 适用场景

- 想快速了解今天或过去 24 小时的重点新闻。
- 想让 AI 同时参考中文和英文来源。
- 想区分事实、各方表态、评论和争议信息。
- 想避免旧新闻冒充最新新闻。

## 完整提示词

```text
请帮我做一份新闻简报。

主题：
【填写：全球政治 / AI / 美股 / 德国新闻 / 东京头条 / 拜仁 / 足球等】

时间范围：
【今天 / 过去24小时 / 本周 / 指定日期】

要求：
1. 优先使用权威来源
2. 标明来源
3. 不要只看中文媒体
4. 如果是国际新闻，尽量包含英文来源
5. 如果是政治新闻，要区分事实、各方表态和评论
6. 如果有争议，要说明不同说法
7. 不要标题党
8. 不要用过期新闻冒充最新新闻

输出格式：
1. 今日最重要的 5-7 条
2. 每条包括：
   - 标题
   - 简短摘要
   - 为什么重要
   - 可能影响
   - 来源
3. 最后给一个总评：
   - 今天的主线是什么
   - 后续最值得关注什么
```

## 输出格式

- 今日最重要的 5-7 条。
- 每条包含标题、摘要、为什么重要、可能影响和来源。
- 最后给总评和后续关注点。

## 注意事项

- 不要编造事实或来源。
- 重要信息需要二次验证。
- 新闻具有时效性，必须标明日期和来源。
- 不要用过期新闻冒充最新新闻。
- 有争议的政治新闻要区分事实、表态和评论。
- 没有可靠来源时，要明确写「无法确认」。
