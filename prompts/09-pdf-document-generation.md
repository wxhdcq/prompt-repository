# PDF/文档生成模板

## 用途

用于生成教程、说明书、报告、PDF、可复制文档和给 Codex 的说明。

## 适用场景

- 想生成一份结构清楚的中文文档。
- 想写教程、使用指南、报告或作业。
- 想让 Codex 生成 PDF 并检查中文字体和版式。
- 想把技术步骤写成可验证的操作路径。

## 完整提示词

```text
请帮我生成一份中文文档。

主题：
【填写主题】

用途：
【教程 / 报告 / 说明书 / 作业 / 给 Codex 的说明 / 使用指南】

要求：
1. 先给结论
2. 结构清楚
3. 适合新手理解
4. 中文自然
5. 不要堆砌术语
6. 有步骤、有注意事项、有常见错误
7. 如果要生成 PDF，必须检查中文字体和版式
8. 如果有表格，要适合手机阅读
9. 如果是教程，要包含完整操作路径
10. 如果是技术文档，要包含命令、文件路径和验收方式

输出结构：
1. 标题
2. 适合谁看
3. 先看结论
4. 详细步骤
5. 常见问题
6. 风险提醒
7. 最后行动清单
```

## 输出格式

- 标题。
- 适合谁看。
- 先看结论。
- 详细步骤。
- 常见问题。
- 风险提醒。
- 最后行动清单。

## 注意事项

- 不要编造事实、数据、截图、引用或实验结果。
- 重要信息需要二次验证。
- 如果生成 PDF，必须检查中文字体、分页、表格和手机阅读体验。
- 技术文档要写清命令、文件路径和验收方式。
- 不要覆盖原始材料，优先生成派生文档。
