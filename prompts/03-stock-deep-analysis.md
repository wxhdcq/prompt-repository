# 股票深度分析模板

## 用途

用于分析单只股票、ETF 或行业，帮助判断短线、中线、长期配置和风险。

## 适用场景

- 想分析 NVDA、AMD、MU、MSTR、QQQM 等标的。
- 想判断是否值得关注、是否过热、是否需要等回调。
- 想把基本面、最新消息、技术面和风险放在一个框架里看。
- 想得到激进、稳健、保守三种方案，但不需要绝对买卖指令。

## 完整提示词

```text
请对以下股票/ETF 做深度分析：

标的：
【填写股票代码，例如 NVDA、AMD、MU、MSTR、QQQM】

我的目的：
【短线交易 / 中线持有 / 长期配置 / 判断是否加仓 / 判断是否止盈】

请按以下结构输出：

1. 先给结论
- 当前是否值得关注
- 适合短线还是中线
- 最大风险是什么
- 最重要的观察价位或事件是什么

2. 基本面
- 公司业务
- 收入来源
- 增长逻辑
- 利润率
- 估值是否偏高
- 与同行相比如何

3. 最新消息
- 财报
- 管理层讲话
- 评级变化
- 监管风险
- 行业新闻
- 宏观影响

4. 技术面
- 趋势
- 支撑位
- 压力位
- 成交量
- 是否过热
- 是否有回调风险

5. 短线策略
- 适合追吗
- 适合等回调吗
- 关键观察位
- 失效条件

6. 中线策略
- 适合分批买吗
- 适合持有吗
- 需要警惕什么

7. 风险
- 估值风险
- 财报风险
- 行业风险
- 政策风险
- 情绪风险

8. 最后给我一个明确结论
请用：
- 激进方案
- 稳健方案
- 保守方案

注意：
不要给绝对化买卖建议。
不要编造实时数据。
所有最新数据都要说明来源或不确定性。
```

## 输出格式

- 先给明确结论和最大风险。
- 再按基本面、消息面、技术面、策略和风险展开。
- 最后给激进、稳健、保守三种观察方案。

## 注意事项

- 投资内容不构成买卖建议。
- 不要编造股价、财报、评级、新闻或实时行情。
- 重要信息需要二次验证。
- 最新数据必须说明来源；查不到就写「数据缺失」或「需二次验证」。
- 不要连接券商、保存账户信息、自动交易或自动下单。
