# 银行卡权益检查模板

## 用途

用于检查中信 GBA 卡、汇丰 Pulse、恒生香港、建行亚洲等银行卡活动、返现、迎新、保级和绑定支付限制。

## 适用场景

- 想确认当前有哪些返现、积分、迎新或保级活动。
- 想知道活动是否需要报名、是否有地区限制。
- 想区分官方条款、社区经验和个人推测。
- 想避免错过截止日或踩中支付限制。

## 完整提示词

```text
请按“银行卡权益检查模式”帮我查这张卡或这些银行活动：

目标：
【填写：中信 GBA 卡 / 汇丰 Pulse / 恒生香港 / 建行亚洲 / 其他】

我关心：
- 新活动
- 返现
- 迎新
- 保级
- 绑定支付
- 云闪付/Apple Pay/支付宝/微信支付限制
- 活动截止日
- 是否需要报名
- 是否有地区限制
- 是否值得参加

请按以下来源优先级：
1. 银行官方网页
2. 官方活动条款 PDF
3. 银行 App 公告
4. 客服说明
5. 社区经验帖
6. 第三方整理文章

请区分：
- 官方确定信息
- 社区经验
- 推测
- 我需要自己进 App 确认的内容

输出：
1. 当前可用活动
2. 即将结束活动
3. 返现/积分规则
4. 参加门槛
5. 是否需要报名
6. 最容易踩坑的地方
7. 对我是否值得
8. 下一步操作清单

注意：
不要把社区经验当官方结论。
不要保证一定成功。
银行卡权益以官方条款和实际账户显示为准。
```

## 输出格式

- 先给是否值得关注和最重要截止日。
- 再按活动表格列出权益、门槛、报名、支付限制和来源。
- 最后列下一步操作清单。

## 注意事项

- 不要编造事实，缺少官方来源时写「需二次验证」。
- 重要信息需要二次验证。
- 银行权益以官方条款和 App 实际显示为准。
- 不要输入银行卡号、身份证件、验证码、密码或完整账单。
- 社区经验只能作为参考，不能替代官方条款。
- 不要保证返现一定成功，尤其是跨境、渠道、MCC 和地区限制。
